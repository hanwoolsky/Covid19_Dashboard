# covid19

## How to Implement?
    $ git clone https://github.com/hanwoolsky/covid19.git
    $ virtualenv env
    $ source env/bin/activate
    $ which python = ~~/python
    $ source ~~/activate
    $ export VIRTUALENVWRAPPER_PYTHON=$(which python3)
    $ source /usr/local/bin/virtualenvwrapper.sh
    $ mkvirtualenv (name)
    $ workon (name)
